Smart Greenhouse Automation System

A smart greenhouse automation system designed for cactus cultivation, integrating IoT sensors with machine learning to monitor environmental conditions and automate greenhouse operations in real time. The system provides intelligent decision support and a responsive dashboard for monitoring and manual control.

Key Features

  • Real-time monitoring of temperature, humidity, and light intensity
  • Automated control of fan, water pump, and grow lights
  • Machine learning-based decision making for environmental control
  • Web-based dashboard for live monitoring and manual overrides
  • Cloud-based data storage for historical analysis

Machine Learning Workflow

  • Sensor data collection and preprocessing
  • Feature engineering using environmental parameters
  • Training machine learning models to predict optimal conditions
  • Model evaluation and tuning for improved accuracy
  • Integration of model outputs into automation logic

Technologies Used

  • Python (scikit-learn) for machine learning model training
  • ESP32 microcontroller for IoT integration
  • DHT22 & TSL2561 sensors for environmental data
  • Node.js backend for data handling and communication
  • React.js frontend for dashboard visualization
  • Firebase (Realtime Database & Firestore) for cloud storage

What This Project Demonstrates

  • End-to-end machine learning system development
  • Integration of ML models with real-world IoT hardware
  • Data-driven automation and decision-making
  • Full-stack development skills (backend, frontend, cloud)
  • Ability to design scalable, real-time intelligent systems
