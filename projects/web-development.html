Server Side Coursework (ServerSideCW)

A server-side web application developed as part of an academic coursework to demonstrate backend development concepts, database interaction, and structured request handling. The project focuses on building a reliable backend system that supports dynamic data operations and clean separation of concerns.

Key Features

  • Backend-driven web application architecture
  • CRUD operations for managing application data
  • Server-side validation and error handling
  • Integration with a database for persistent storage
  • Structured and modular server-side codebase

Tech Stack

  • Server-side programming (Java / Node.js – coursework-based)
  • HTML, CSS, JavaScript
  • Database integration (MySQL or equivalent)
  • Git & GitHub for version control

PLM Color Automation

A web-based automation solution developed to streamline PLM color-related workflows. The application reduces repetitive manual tasks by automating color creation and processing logic, improving accuracy and operational efficiency for business users.

Key Features

  • Automation of PLM color management workflows
  • Reduction of manual data entry and human errors
  • User-friendly interface for quick operations
  • Structured input validation and processing
  • Production-ready deployment for business use

Tech Stack

  • Frontend: Angular / Next.js
  • Automation & Logic: JavaScript
  • Collaboration with functional consultants
  • Deployed to live production environment
