Skin Consultation Manager

A Java-based desktop application developed to manage the operations of a skin consultation centre. The system enables efficient management of doctors, patients, and consultation bookings while using JSON files for persistent data storage without a traditional database.

Key Features

  • Doctor profile management with specialization and availability
  • Patient registration and record maintenance
  • Consultation booking and scheduling system
  • Persistent data storage using structured JSON files
  • Object-oriented design for scalability and maintainability

Tech Stack

  • Java
  • Object-Oriented Programming (OOP)
  • JSON file handling

Cruise Ship Boarding System

A Java-based system designed to manage cruise ship cabin bookings and passenger boarding. The application allows users to check cabin availability, allocate cabins efficiently, and place passengers into a waiting queue when all cabins are fully booked.

Key Features

  • Cabin availability tracking and booking
  • Passenger registration and boarding management
  • Waiting queue system when cabins are full
  • Efficient handling of booking edge cases
  • Well-structured OOP-based system design

Tech Stack

  • Java
  • Object-Oriented Programming (OOP)
  • Queue-based data structures
